# pileless-sdk-ts

> TypeScript SDK for the Pile protocol.

**Status:** placeholder. Public RFC and reference implementation are in active development.

## What is Pileless?

Pileless is an open protocol for agent-to-human decision capture. Agents accumulate **Piles** of work items (variants, drafts, exceptions) that need a human decision. Humans review, approve, reject, or revise on their own schedule. Audit-log everything.

- Spec: [pile-protocol](https://github.com/pileless/pile-protocol)
- Reference MCP server: [pileless-mcp](https://github.com/pileless/pileless-mcp)
- TypeScript SDK: [pileless-sdk-ts](https://github.com/pileless/pileless-sdk-ts)
- Python SDK: [pileless-sdk-py](https://github.com/pileless/pileless-sdk-py)
