# Contributing

Thanks for considering a contribution.

## Where decisions get made

- **Protocol changes** — discussed in [pile-protocol/issues](https://github.com/pileless/pile-protocol/issues) as RFCs. Changes to the wire format, schema, or semantics MUST land there first.
- **Implementation changes** — PRs welcome on the relevant SDK or reference server. Track conformance to the latest RFC.
- **Bugs** — file in this repo's issues with a minimal repro.

## Code of Conduct

Be excellent to each other. Don't be a jerk in issues, PRs, or discussions. Maintainers reserve the right to remove anything (or anyone) violating that, no further explanation owed.
